## ANVIL UI
An admin UI for foundry's anvil.
Account management, block explorer/mangament. 

Built on [viem](https://github.com/wagmi-dev/viem)

## Installation

Clone repository
```bash
git clone git@github.com:literallymarvellous/anvil-ui.git
```
Install dependencies
```bash
npm install
```
Start Anvil and UI
```bash
npm run dev
```
